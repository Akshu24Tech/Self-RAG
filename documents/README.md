# Documents Folder

Place your PDF documents here.

## Instructions

1. Copy any PDF files into this folder
2. The Self-RAG system will automatically load all PDFs
3. Ask questions about your documents!

## Examples

```
documents/
├── research_paper_1.pdf
├── research_paper_2.pdf
├── company_policy.pdf
├── user_manual.pdf
└── ...
```

## Supported Formats

- ✅ PDF files (.pdf)
- ❌ Other formats not supported yet

## Tips

- Use descriptive filenames
- Group related documents together
- Remove unnecessary PDFs to improve performance
